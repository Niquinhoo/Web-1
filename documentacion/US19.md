# User Story #19 - Buscador de productos

**Codigo:** `#main-s2-us19`

**Nombre:** Buscador de productos

### Detalle

La historia pide buscar productos por nombre desde el formulario del header.

### Implementacion realizada

Servicio:

- `searchProductsByName(query)` en `services/productsService.js`
- filtra por coincidencia parcial sobre `title` (case-insensitive)

Ruta:

- `GET /search?query=...` en `routes/search.router.js`

Header:

- `views/partials/molecules/home/search-bar.ejs` se cambio a formulario `GET` con `action="/search"` y campo `query`.

Vista de resultados:

- `views/pages/search/search-results-page.ejs`
- `views/partials/pages/search-results-content.ejs`
- si no hay resultados, se muestra mensaje amigable.

### Validacion

Cumplido:

- Escenario `/search?query=camisa` soportado.
- Se filtra por coincidencia parcial.
- Se muestra mensaje cuando no hay resultados.
- No se usa API.
- No se usa DB.
