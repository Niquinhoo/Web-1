# User Story #14 - Layout base

**Codigo:** `#main-s2-us14`

**Nombre:** Layout base

### Detalle

La historia pide un layout compartido para todas las paginas (excepto login y registro), para mantener una experiencia uniforme.

### Implementacion realizada

Se creo el layout principal:

- `views/layouts/main.ejs`

Este layout centraliza:

- estructura HTML base
- fuentes y estilos comunes
- header/navbar comun
- footer comun
- render del contenido por parcial dinamico

Se adaptaron las vistas para que usen `main.ejs` (excepto login y register):

- home
- category
- product detail
- product not found
- cart
- checkout
- account
- 404
- 500

Para mantener orden, se agregaron parciales de contenido en:

- `views/partials/pages/*.ejs`

### Validacion

Cumplido:

- Existe `views/layouts/main.ejs`.
- Las vistas principales usan layout compartido.
- Login y Register se mantienen fuera del layout, como pide la story.
- No se usa API.
